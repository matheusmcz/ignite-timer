# ignite-timer

This aplication was build with:
- Vite
- React;
- TypeScript;
- React-router-dom;
- React Hook Form;
- Context API;
- Styled Components;

The goal is to improve my skills with these technologies.

To use this app:
```Git clone git@github.com:matheusmcz/ignite-timer.git```

```Yarn``` or ```npm i``` (to install all deps)
